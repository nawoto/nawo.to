まず boehmgc6.8 を取ってきて、こんな感じでビルド。

```
% ./configure CFLAGS="-D__DARWIN_UNIX03=0" --enable-threads=posix \
--enable-parallel-mark \
--enable-cplusplus \
--disable-shared \
--enable-static
% make
% DESTDIR=~/dest make install
```

で、あとはこれを static link してビルドするだけ。
